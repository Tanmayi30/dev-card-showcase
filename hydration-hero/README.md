# Hydration Hero

A browser extension that reminds users to drink water, tracks daily intake, and uses AI to suggest optimal hydration times based on activity and weather.

## Features
- Log water intake
- View daily progress
- AI-based hydration suggestions
- Set reminders to drink water

## Usage
1. Load the extension in your browser (Developer mode > Load unpacked)
2. Click the extension icon to open the popup
3. Log your water intake and set reminders
