# Focused Reading Mode

A browser extension that transforms any webpage into a distraction-free, readable format, with customizable fonts, colors, and ambient sounds.

## Features
- Removes distractions from webpages
- Customizable fonts and background colors
- Ambient sound options (rain, forest, waves, cafe)
- Easy activation from popup

## Usage
Install the extension, click the popup, and activate reading mode. Customize your experience with the settings panel.
